## Summary
This project is a responsive Social Profile Card Component. It includes a profile section with an avatar and info, as well as social media links with on-hover responsiveness.
I built a vanilla HTML/CSS version plus a React + Vite version to practice component-based architecture.

## Tech Stack
- Vanilla HTML/CSS
- React + Vite

## Installation
1. Clone the repo: https://github.com/jc1471/fm-social-links-profile.git
2. Navigate to the react folder: cd react
3. Install dependencies: npm install
4. Run development server: npm run dev
